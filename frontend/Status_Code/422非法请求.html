非法请求

你的请求包含恶意内容、非法参数或不符合API调用规则，已被安全系统拦截

可能的拦截原因

  • 请求参数包含SQL注入、XSS等恶意字符（如{{malicious_param}}）
  • 请求头伪造、Referer来源非法或User-Agent异常
  • 图片路径/分类名称包含敏感字符或非法格式
  • 重复提交相同的恶意请求，触发安全防护规则
  • IP地址在恶意请求黑名单中

合规调用指引

✅ 仅使用合法字符（字母、数字、中文、下划线、斜杠）作为参数

✅ 遵守API调用规则，单次请求仅获取一张图片

✅ 使用真实的User-Agent，不伪造请求来源

✅ 图片路径格式示例：二次元/123.webp

✅ 分类名称仅使用已公布的合法分类（Donate、二次元、人物等）
